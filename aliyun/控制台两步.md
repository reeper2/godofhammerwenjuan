# 在 Firefox 里完成这两步（约 3 分钟）

当前 AccessKey **不能自动创建 Bucket**，需用控制台操作一次。

## 步骤 1：创建 OSS 存储桶

1. 打开：https://oss.console.aliyun.com/bucket?region=cn-hangzhou  
2. **创建 Bucket**  
   - 名称：`gsl-survey-hz-moon`（须与 `.env` 一致）  
   - 地域：**华东1（杭州）**  
   - 读写权限：**私有**  

## 步骤 2：给 RAM 用户授权

1. 打开：https://ram.console.aliyun.com/users  
2. 找到创建 AccessKey 的那个用户 → **添加权限**  
3. 选择 **自定义策略** → 创建策略 → 粘贴 `aliyun/ram-oss-policy.json` 内容  
4. 或直接勾选 **AliyunOSSFullAccess**（测试用，权限较大）

## 完成后在终端执行

```bash
cd "/Users/moon/Pool/01_Projects/2605_gsl问卷公网发送"
python3 aliyun/setup.py --bucket gsl-survey-hz-moon \
  --access-key-id '你的ID' --access-key-secret '你的Secret'
```

（`.env` 已写好密钥，也可只跑 `--bucket gsl-survey-hz-moon` 若环境变量已加载）

函数计算：上传 `aliyun/fc-submit/fc-submit.zip`，环境变量见 `aliyun/部署说明.md`。

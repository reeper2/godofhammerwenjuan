# 阿里云问卷收数 — 函数计算 + 对象存储 OSS

问卷网页（GitHub Pages 等）通过 **HTTP 公网地址** 提交答卷；函数计算将每条记录存为 OSS 上的 JSON 文件。密钥只放在阿里云函数环境变量里，**不会**出现在网页源码中。

## 架构

```
受访者浏览器  --POST JSON-->  函数计算（HTTP 触发器）
                                    |
                                    v
                              OSS 存储桶 / survey-responses/*.json

管理员本机  --OSS SDK-->  下载全部 JSON  -->  export_report.py
```

## 地域说明（本项目：杭州）

| 项目 | 取值 |
|------|------|
| 地域 | **华东1（杭州）** |
| OSS Endpoint | `oss-cn-hangzhou.aliyuncs.com` |
| 函数计算 | 创建函数时地域选 **杭州**，HTTP 地址一般为 `*.cn-hangzhou.fcapp.run` |

> OSS Bucket 与函数计算 **必须在同一地域**（均选杭州），否则函数无法写 OSS。

## 一、开通 OSS 存储桶

1. 登录 [阿里云控制台](https://oss.console.aliyun.com/)  
2. **创建 Bucket**  
   - 地域：**华东1（杭州）**  
   - 存储类型：标准存储  
   - 读写权限：**私有**（不要选公共读）  
3. 记下 **Bucket 名称**；Endpoint 使用 `oss-cn-hangzhou.aliyuncs.com`

## 二、创建 RAM 子账号（推荐）

1. [RAM 控制台](https://ram.console.aliyun.com/) → 用户 → 创建用户  
2. 勾选「OpenAPI 调用访问」，保存 **AccessKey ID** 和 **AccessKey Secret**  
3. 为用户授权自定义策略（仅访问本 Bucket）：

```json
{
  "Version": "1",
  "Statement": [
    {
      "Effect": "Allow",
      "Action": ["oss:PutObject", "oss:GetObject", "oss:ListObjects"],
      "Resource": [
        "acs:oss:*:*:你的Bucket名称",
        "acs:oss:*:*:你的Bucket名称/survey-responses/*"
      ]
    }
  ]
}
```

> 函数计算写入 OSS、你本机导出答卷，共用这一对 Key 即可（保存在函数环境变量与本机 `.env`）。

## 三、部署函数计算

### 3.1 打包代码

在项目根目录执行：

```bash
cd aliyun/fc-submit
pip3 install -r requirements.txt -t ./package
cp index.py ./package/
cd package && zip -r ../fc-submit.zip . && cd ..
```

得到 `aliyun/fc-submit/fc-submit.zip`。

### 3.2 创建函数（函数计算 FC 3.0）

1. 打开 [函数计算控制台](https://fcnext.console.aliyun.com/)，右上角地域选 **华东1（杭州）**  
2. **创建函数** → 事件函数 → 运行环境 **Python 3.10**  
3. 上传 `fc-submit.zip`，**请求处理程序** 填：`index.handler`  
4. **内存** 256 MB，**超时** 30 秒  

### 3.3 配置环境变量

在函数 **配置 → 环境变量** 中添加：

| 变量名 | 示例值 |
|--------|--------|
| `OSS_ENDPOINT` | `oss-cn-hangzhou.aliyuncs.com` |
| `OSS_BUCKET` | `你的bucket名` |
| `OSS_ACCESS_KEY_ID` | RAM 的 ID |
| `OSS_ACCESS_KEY_SECRET` | RAM 的 Secret |
| `OSS_PREFIX` | `survey-responses/` |
| `SURVEY_TOKEN` | （可选）留空则不校验；填写则需与 `config.js` 中一致 |

### 3.4 创建 HTTP 触发器

1. 函数详情 → **触发器** → 创建 → **HTTP 触发器**  
2. 请求方法：**POST**（若控制台支持，同时勾选 OPTIONS，用于浏览器跨域）  
3. 鉴权方式：**匿名**（无需登录即可填问卷）  
4. 复制 **公网访问地址**，形如：  
   `https://survey-xxxxx.cn-hangzhou.fcapp.run`  

若仅支持 POST、提交报 CORS 错误：在触发器或「函数配置」中开启 **允许跨域**，或增加 OPTIONS 方法后重新部署。

## 四、配置问卷网站

编辑项目根目录 `config.js`：

```javascript
window.SURVEY_CONFIG = {
  aliyun: {
    submitUrl: 'https://你的函数HTTP地址',  // 不要漏写 https
    surveyToken: ''  // 若函数设置了 SURVEY_TOKEN，这里填相同字符串
  },
  consoleUrl: 'https://oss.console.aliyun.com/bucket/oss-cn-hangzhou/你的bucket/object?path=survey-responses%2F'
};
```

`git add config.js && git commit && git push`，等待 GitHub Pages 更新。

## 五、本机导出答卷

```bash
cp .env.example .env
# 编辑 .env，填入 OSS 与 AccessKey（与函数环境变量相同）

pip3 install oss2 requests
python3 scripts/fetch_submissions.py
python3 scripts/export_report.py output/survey_results.json -o output/汇总报告.md
```

也可在 OSS 控制台 **文件列表** 中浏览 `survey-responses/`，批量下载 JSON。

- 函数计算：有免费额度，按调用次数计费  
- 新用户可在阿里云「免费试用」中领取 OSS / 函数计算体验金  

## 常见问题

**Q：GitHub Pages 要改吗？**  
不用，只改 `config.js` 里的 `submitUrl`。

**Q：如何防刷？**  
可设置 `SURVEY_TOKEN`；或在 API 网关前加 WAF / 限流（进阶）。

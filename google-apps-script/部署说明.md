# Google 表格数据收集 — 部署步骤

问卷提交后会写入 Google 表格，便于导出 Excel、做图表和撰写报告。

## 1. 新建表格

1. 打开 [Google 表格](https://sheets.google.com)，新建空白工作簿  
2. 将工作簿命名为「AI数字化转型问卷-答卷」  
3. 复制浏览器地址栏中的表格 ID（`/d/` 与 `/edit` 之间的字符串）

## 2. 绑定 Apps Script

1. 在表格中：**扩展程序 → Apps Script**  
2. 删除默认的 `function myFunction()` 内容  
3. 将本目录 `Code.gs` 的全部内容粘贴进去，保存项目  
4. 菜单 **运行 → 运行函数 → setupAdminKey**（首次授权 Google 账号）  
5. 打开 **项目设置 → 脚本属性**，将 `ADMIN_KEY` 改为**强密码**（与网站 `config.js` 中 `adminKey` 一致）

## 3. 部署 Web 应用

1. 点击 **部署 → 新建部署**  
2. 类型选择 **Web 应用**  
3. 设置：
   - 执行身份：**我**
   - 访问权限：**任何人**（填写问卷的人无需登录 Google）
4. 部署后复制 **Web 应用 URL**（形如 `https://script.google.com/macros/s/xxxx/exec`）

## 4. 填写网站配置

编辑项目根目录 `config.js`：

```javascript
window.SURVEY_CONFIG = {
  submitUrl: 'https://script.google.com/macros/s/你的真实ID/exec',
  adminKey: '与脚本属性 ADMIN_KEY 相同',
  sheetUrl: 'https://docs.google.com/spreadsheets/d/你的表格ID/edit'
};
```

保存后 `git commit` 并 `git push`，GitHub Pages 会在 1–2 分钟内更新。

## 5. 验证

1. 浏览器打开问卷链接，填写并提交  
2. 在 Google 表格 `responses` 工作表应出现新行  
3. 在问卷页点击「查看已提交问卷结果」，输入 `adminKey` 应能看到列表  

## 导出与报告

- **直接分析**：在 Google 表格中筛选、数据透视、图表  
- **导出 CSV**：文件 → 下载 → CSV，然后运行  
  `python3 scripts/export_report.py 下载的文件.csv -o output/汇总报告.md`

## 访问说明（国内受访者）

- GitHub Pages 与 Google 服务在国内可能偶发较慢或需科学上网  
- 若受访者无法提交，可改用学校/单位服务器托管静态页，**仍使用同一 `config.js` 与 Apps Script URL**
